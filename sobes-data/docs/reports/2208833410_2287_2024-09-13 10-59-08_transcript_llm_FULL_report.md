## Блок 4: Информация о компании/команде

### Ключевые детали:
- Компания: команда из банковского проекта импортозамещения, связанных с ВТБ.
- Проект: импортозамещение старой монолитной банковской системы (~20 лет), перевод Oracle → Postgres, React фронт вместо Delphi.
- Стек: React 18, TypeScript, Redux Toolkit, Style Components, webpack, микрофронты, собственная дизайн-система AdmiralDFs.
- Архитектура: переход с монолита на микросервисы, использование микрофронтов.
- Команда: 3 команды по 11-12 человек сейчас, планируется до 8 команд.
- Процессы: релизы через ветки git, pre-commit хуки с ESLint, daily делики по 15-20 минут.
- Оборудование: кандидат будет работать на технике банка с ограничениями безопасности.
- Инструменты управления задачами: внутри банка заменили Jira и Confluence на собственные аналоги.
- Особенности: сроки сжаты — два года на импортозамещение, MVP в текущем году (~10% функционала), полный релиз — следующий год.
- Работа: full-time, не аутстафф, фронтенд-разработка банковских форм и финансовых операций.
- Коммуникация: дружная атмосфера, частые шутки и поддержка в командах.
